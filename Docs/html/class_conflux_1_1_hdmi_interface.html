void(*)(const char *currentProcess)  currentProcess,
void(*)(int percentageComplete)  percentComplete,
void(*)(const char *errorMessage)  errorMessage,
void(*)(bool flashSuccessful)  updateComplete,
const char *  pathToFirmware = "" 
)
pure virtual

Calling this function with a valid update source will begin the process of updating the firmware. This is an async process and updates to the process are provided via callbacks.

Parameters
updateSourceEnumeration of possible update sources. Indexed by Conflux::UpdateSource.
currentProcessCallback that provides updates on the current update process.
percentCompleteCallback that provides the current percentage complete.
errorMessageCallback that provides any errors encountered during the process.
updateCompleteCallback that notifies that the update has completed successfully, or otherwise.
pathToFirmwareAllows the location of the update to be provided by absolute path, if the update source is Conflux::UpdateSource::WORKING_DIRECTORY.
Returns
true if the Process was started successfully.
false otherwise.
